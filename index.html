Acerca de AKEOG

Akeog es un equipo de talentos en tecnologías Microsoft, especializados en soluciones IT para proveer a las organizaciones de cualquier tamaño altos estándares de calidad en los servicios de infraestructura, seguridad y servicios en la nube.


Nuestra experiencia nos respalda, brindando toda la confianza que se necesita, enmarcadas con las mejores prácticas, confidencialidad, integridad y disponibilidad de la información sellados por nuestros valores y ética profesional. La simplicidad de nuestros procesos ayuda a cumplir satisfactoriamente con los tiempos propuestos y sacando los mejores resultados.


Somos una empresa privada en constante crecimiento. Nos hemos posicionado como socios de negocios preferenciales ofreciendo soluciones en diferentes industrias tales como Microsoft y Kaspersky, entre otras.

SECURITY AND IDENTITY

  • ANALISIS DE VULNERABILIDADES: Identificar errores de configuración o ausencia de actualizaciones que conlleven a exponer la información confidencial de la organización.
    1. INGENIERIA SOCIAL: Evaluar los usuarios de su compañía mediante diferentes técnicas obteniendo información confidencial o privilegiada.
  • ETHICAL HACKING: Conocer la infraestructura de tal manera que se pueda identificar los elementos vulnerables y realizar las respectivas correcciones.
  • AUDITORIA DE SEGURIDAD DE LA INFORMACIÓN: Proteger sus activos informáticos y procedimientos críticos del negocio aplicando las buenas prácticas de la industria como la norma ISO 27001:2013.
  • CIFRADO DE DATOS: Protección de los datos para evitar la fuga o robo de la información en los equipos de cómputo, discos, particiones o almacenamientos extraíbles utilizando tecnologías como:
    1. Microsoft Bitlocker
    2. Kaspersky Data Encryption
    3. Symantec Endpoint Encryption
    4. McAfee Endpoint Encryption
  • IDENTITY AND ACCESS: Administra la comunicación entre usuarios y dominios, incluyendo procesos de inicio de sesión de usuarios, autenticación y búsqueda de directorio con las tecnologías de:
    1. Microsoft Active Directory Domain Services (AD DS)
    2. Microsoft Azure Active Directory Services (AADS)
    3. Microsoft Active Directory Lightweight Directory Services (AD LDS)
    4. Microsoft Identity Manager (MIM)

VIRTUALIZATION

  • VIRTUALIZACIÓN DE SERVIDORES: Administra y agiliza la disponibilidad de los recursos de cómputo para sus aplicaciones más exigentes permitiendo de manera rápida la consolidación de recursos de hardware necesarios basadas en las siguientes tecnologías:
    1. Microsoft Hyper-V
    2. VMware vSphere, ESXi
  • VIRTUALIZACIÓN DE PERFILES DE USUARIO: Captura y centraliza las configuraciones de las aplicaciones y del sistema operativo Windows para cada perfil de usuario, con los parámetros deseados para cada aplicación con:
    1. Microsoft User Experience Virtualization (UE-V)
  • VIRTUALIZACIÓN DE APLICACIONES: Permite a los administradores implementar, actualizar y soportar aplicaciones como servicios en tiempo real. Las aplicaciones son transformadas desde un producto local a un servicio de administración centralizada, permitiendo así evitar configuraciones adicionales en el equipo o sistema operativo:
    1. Microsoft Application Virtualization (App-V)
  • CONTENEDORES: ¡Implementa aplicaciones en segundos! gracias a que dentro del mismo kernel de sistema operativo se ejecutan muchas aplicaciones dentro de paquetes aislados y con sus únicas dependencias permitiendo que cada aplicación consuma pocos recursos comparado con las máquinas virtuales.
    1. Windows Server 2016 Containers
    2. Microsoft Azure Containers

DISASTER RECOVERY AND HIGH AVAILABILITY

  • FAILOVER CLUSTERING: Proporciona alta disponibilidad y escalabilidad entre servidores permitiendo equilibrar cargas de trabajo, respaldando los servicios y aplicaciones como DHCP, Hyper-V, SQL Server, Exchange Server, File Server entre otras.
    1. Windows Server Failover Clustering (WSFC)
  • COPIAS DE SEGURIDAD: Respaldo de la información tanto locales como en la nube con eficientes copias de seguridad fiables y/o cifradas, usando tecnologías como:
    1. Microsoft Azure Backup (MAB)
    2. Windows Server Backup
    3. System Center Data Protection Manager (SCDPM)
  • DISASTER RECOVERY: Automatiza la protección y recuperación de sus servicios o aplicaciones con un centro de respaldos de la nube de manera rápida, garantizando la disponibilidad y recuperación.
    1. Azure Site Recovery

CLOUD PRODUCTIVITY

  • EXCHANGE ONLINE: Administra y protege su información del correo electrónico sin requerir hardware y de manera fácil. Con funciones avanzadas de seguridad, control y permitiendo tener su entorno de correo electrónico en los servidores de Microsoft de tal forma garantizando un 99,9% de tiempo de actividad.
  • SHAREPOINT ONLINE: Crea de manera rápida y sencilla sitios de grupos y portales para publicar, compartir y trabajar de forma colaborativa el contenido de su organización con compañeros de trabajo dentro y fuera desde cualquier dispositivo.
  • SKYPE FOR BUSINESS ONLINE: Se obtiene a través de una sola aplicación con Office los beneficios de mensajería, reuniones y uso compartido de la pantalla con los miembros de la organización a través de cualquier dispositivo. El servicio quedara alojado en los servidores de Microsoft permitiendo minimizar al máximo muchas de la tareas operacionales y administrativas que este requiera.
  • OFFICE ONLINE: Vea, edite y comparta archivos de Word, Excel, PowerPoint y OneNote de manera eficaz a través de una sola interfaz de usuario conocida.

ENTERPRISE MOBILITY SUITE

  • CONTROL DE IDENTIDAD Y ACCESO: Administrar de forma centralizada las identidades a través de una forma segura con un único inicio de sesión en sus aplicaciones dentro de su centro de datos y nube.
    1. Azure Active Directory Services (AADS)
  • ADMINISTRACION DE DISPOSITIVOS Y APLICACIONES: Brindar administración de dispositivos (Mobile Device Management) y aplicaciones (Mobile Application Management) con el fin de mantener su información segura e integrándose con su entorno local y en la nube a través de una sola consola.
    1. Microsoft Intune (MDM y MAM)
  • PROTECCION DE LA INFORMACION: Clasifica su información de acuerdo al nivel de confidencialidad que exige la organización y se aplica protección de la tal forma que permite compartir los datos de forma segura, adquiriendo visibilidad y control de los mismos.
    1. IRM (Information Rights Management)
  • SEGURIDAD BASADA EN IDENTIDAD: Protege los accesos de su organización con inicios de sesión únicos, por medio de autenticación Multi-Factor (2FA) y acceso condicional basado en el riesgo, garantizando una identidad común y perdida de datos.
